# 第十章：初始配置：四柱架构解析

**Chapter 10: Initial Configuration: Four Pillars Architecture**

## 10.1 用户实例：日元作为相对论参考系

八字命理（BaZi）不仅是预测术，更像是一个**实体组件系统（Entity Component System - ECS）**。

### 10.1.1 观察者的确立
在四柱（年、月、日、时）这 8 个字中，为什么选**日干**代表“我”？
*   年柱是公转周期，代表宏观背景（国家、祖先）。
*   日柱是自转周期，代表个体的昼夜节律。

确立日元，就是确立**坐标原点 (0,0,0)**。
如果没有这个原点，所有的生克关系都是无意义的物理现象。只有确立了“我”，金克木才变成了“官鬼克身”（压力）或“我克妻财”（掌控）。
这完全符合**相对论**：物理定律的形式依赖于参考系的选择。

## 10.2 环境参数：月令权重

在八字算法中，**月令（出生月份的地支）** 拥有最高的权重参数（Weight）。
通常占据 40% - 50% 的算力。

### 10.2.1 初始温度设定 (Temperature Setting)
月令决定了系统的**基调**。
*   生于午月（夏天）：系统处于高能、高热状态。急需水（调候）来降温。
*   生于子月（冬天）：系统处于低能、冻结状态。急需火（调候）来激活。

这不仅仅是五行生克，这是**热力学环境**。
一个脱离了月令环境的八字分析，就像是在真空中讨论空气动力学，是完全错误的。
道家模型极其重视**时令（Timing）**，即系统所处的**大环境周期阶段**。
