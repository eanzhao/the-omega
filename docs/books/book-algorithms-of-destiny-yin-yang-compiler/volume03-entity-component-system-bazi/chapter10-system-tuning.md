# 第十章：系统调优：格局与用神

**Chapter 10: System Tuning: Patterns and Useful God**

## 10.1 损失函数：旺衰平衡与格局顺逆的数学本质

### 10.1.1 扶抑格：凸优化问题 (Convex Optimization)
对于大多数普通八字（正格），系统的目标是**追求平衡 (Equilibrium)**。
这等价于寻找一个凸函数的全局最小值。
*   损失函数 $L = |Energy_{self} - Energy_{environment}|$。
*   我们希望 $L \to 0$。
*   如果身弱，策略是加强自我（印比）；如果身强，策略是削弱自我（克泄耗）。

### 10.1.2 从格：非凸优化与相变
对于特殊格局（从格），系统发生**相变 (Phase Transition)**。
能量极度偏向一方，平衡态不再是最低能级。系统必须**坍缩**到一个新的稳态 —— 顺从大势。
这是一种**过阻尼 (Over-damped)** 或 **逃逸 (Runaway)** 模式。
在此模式下，对抗（追求平衡）会导致巨大的内耗，顺从才是**最优传输路径 (Optimal Transport Path)**。

## 10.2 梯度下降：取用神就是在寻找人生最优化的方向矢量

**用神 (Useful God)** 是整个命理算法的核心。
它是能让系统**总能量耗散最小、做功效率最大**的那个关键变量。

### 10.2.1 梯度方向 (Gradient Direction)
八字就像一个复杂的高维地形图。
取用神，就是计算当前的**梯度向量 $-\nabla L$**。
*   如果命局“火炎土燥”，梯度指向“水”。
*   如果命局“金寒水冷”，梯度指向“火”。

**改命**，就是在人生的每一步决策中，都有意识地沿着**用神指示的梯度方向**移动。
*   选择用神代表的职业。
*   居住在用神代表的方位。
*   培养用神代表的性格特征（如用神为印，就多读书；用神为食伤，就多表达）。

这就是**基于物理梯度的命运优化算法**。
